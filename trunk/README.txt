A set of arduino libraries has been created for I2C devices
They must be synced with your Arduino  IDE:
	arduino-00xx\hardware\libraries	

NOTE:Arduino uses the 7 bits format so I2C devices must be addressed 
with the hex addr right shift {HEX} >> 1
